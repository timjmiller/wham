# Articles

### Overview

These vignettes provide an introduction to fitting state-space
assessment models using the **WHAM** package. They get progressively
more complex, so we advise working through them in order unless you are
only interested in a specific topic.  
  
To run ALL examples (takes ~1 hour):  
`# setwd("choose/where/to/save/output") # otherwise in working directory`  
[`library(wham)`](https://timjmiller.github.io/wham/)  
`wham.dir <- find.package("wham")`  
`source(file.path(wham.dir, "example_scripts", "run_all_examples.R"))`
